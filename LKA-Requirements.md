## 1. 概述

### 1.1 功能定义

LKA（Lane Keeping Assist，车道保持辅助）是一种 L2 级辅助驾驶功能，通过控制方向盘力矩帮助车辆保持在车道内，防止无意识的车道偏离。

### 1.2 功能定位

- **级别**: L2 级辅助驾驶基础功能
- **组合**: 与 ACC（自适应巡航）、ELK（紧急车道保持）协同工作
- **目标**: 提升驾驶安全性，减轻驾驶员负担

### 1.3 功能范围

- **LKA**: 基础车道保持功能
- **ELK**: 紧急车道保持（防止偏离碰撞）
- **协同**: LKA + ELK 组合方案

## 2. 功能需求

### 2.1 激活条件

#### 2.1.1 系统激活

- **默认状态**: 关闭
- **激活方式**: 驾驶员通过按键/菜单手动开启
- **状态记忆**: 记住上次状态，下次启动自动恢复
- **激活指示**: 仪表盘显示 LKA 图标（绿色 = 激活，灰色 = 待机）

#### 2.1.2 功能激活条件

**必要条件**（全部满足）:
1. 车速 ≥ 60 km/h
2. 车速 ≤ 120 km/h
3. 车道线清晰可见（置信度 > 70%）
4. 驾驶员手在方向盘（通过力矩传感器检测）
5. 无系统故障
6. 道路曲率半径 > 250m

**激活流程**:
```
系统启动 → 驾驶员激活 LKA → 检测激活条件 → 
满足条件 → LKA 功能激活 → 仪表盘显示绿色图标
```

---

### 2.2 功能运行

#### 2.2.1 感知系统

**传感器配置**:
- **前置摄像头**: 识别车道线、道路边界
- **前向毫米波雷达**: 提供距离信息、目标检测
- **融合策略**: 视觉为主，雷达辅助验证

**感知要求**:
- **车道线识别率**: ≥ 95%（高速公路）
- **识别距离**: 0-80m
- **更新频率**: 20Hz
- **置信度要求**: > 70% 才能激活

#### 2.2.2 控制策略

**控制方式**: 力矩控制（Torque Control）

**控制逻辑**:
```
感知车道偏差 → 计算目标力矩 → EPS 电机施加力矩 → 
方向盘修正 → 车辆回到车道中心
```

**力矩参数**:
- **最大力矩**: ≤ 3 Nm（符合 UN-R79）
- **力矩变化率**: ≤ 0.5 Nm/s（舒适性要求）
- **控制频率**: 100Hz

**控制范围**:
- **横向偏差**: ±0.5m（车道中心为基准）
- **航向角偏差**: ±5°
- **控制目标**: 保持在车道中心 ±0.2m

#### 2.2.3 速度范围

- **激活速度**: 60-120 km/h
- **最佳工作速度**: 80-100 km/h
- **降级区间**: 
  - 60-70 km/h: 降低辅助力度（50%）
  - 110-120 km/h: 降低辅助力度（50%）

---

### 2.3 退出条件

#### 2.3.1 分级退出机制

**Level 1: 警告阶段**（车道偏离风险）
- **触发条件**: 横向偏差 > 0.3m 或航向角 > 3°
- **警告方式**: 
  - 视觉：仪表盘黄色图标 + 文字提示
  - 声音：1次提示音（500ms）
- **持续时间**: 3秒
- **系统状态**: LKA 仍激活

**Level 2: 降级阶段**（即将退出）
- **触发条件**: 
  - 警告后 3 秒仍未回正
  - 或车道线丢失
  - 或速度超限
- **降级方式**:
  - 视觉：仪表盘橙色图标 + 文字提示
  - 声音：连续提示音（1Hz，3次）
  - 功能：辅助力度降至 50%
- **持续时间**: 5秒
- **系统状态**: LKA 降级运行

**Level 3: 退出阶段**（完全退出）
- **触发条件**: 降级后 5 秒仍无法恢复
- **退出方式**:
  - 视觉：仪表盘灰色图标 + 文字提示"LKA 已退出"
  - 声音：长提示音（1s）
  - 功能: 完全停止辅助
- **系统状态**: LKA 待机（需手动重新激活）

#### 2.3.2 驾驶员接管

**接管逻辑**:
- **检测方式**: EPS 力矩传感器检测方向盘手力输入
- **阈值**: > 0.5 Nm 持续 > 0.5s
- **响应**: 暂停 LKA 控制，保持系统激活
- **恢复**: 驾驶员松手后 1 秒自动恢复

**接管优先级**: 驾驶员 > 系统

---

### 2.4 故障处理

#### 2.4.1 故障检测

**感知故障**:
- 摄像头遮挡/污损
- 车道线识别失败（连续 10 帧）
- 传感器通信异常
- 数据超时（> 100ms）

**系统故障**:
- EPS 通信异常
- 控制器故障
- 电源异常

#### 2.4.2 故障响应

**警告后退出**（C 方案）:
1. **故障检测** → 系统识别故障
2. **立即警告** → 仪表盘红色图标 + 文字提示
3. **5 秒过渡** → 驾驶员接管时间
4. **功能退出** → LKA 完全停止
5. **状态记录** → DTC 故障码记录

**故障码定义**:
- `P0401`: 摄像头信号丢失
- `P0402`: 车道线识别异常
- `P0403`: EPS 通信故障
- `P0404`: 控制器内部故障

---

### 2.5 人机交互（HMI）

#### 2.5.1 状态指示

**仪表盘显示**:
- **灰色图标 + "LKA 待机"**: 系统关闭
- **绿色图标 + "LKA 激活"**: 功能运行中
- **黄色图标 + "请回正方向"**: 警告阶段
- **橙色图标 + "LKA 降级"**: 降级运行
- **红色图标 + "LKA 故障"**: 故障状态

**提示音**:
- **激活音**: 1 短音（200ms）
- **警告音**: 1 长音（500ms）
- **退出音**: 1 长音（1s）

#### 2.5.2 控制按键

**物理按键**（方向盘）:
- **短按（< 1s）**: 激活/取消 LKA
- **长按（> 2s）**: 进入 LKA 设置菜单

**菜单设置**:
- LKA 开关
- 辅助力度调节（低/中/高）
- 警告音开关

---

### 2.6 数据记录

#### 2.6.1 事件记录

**触发记录的事件**:
1. 系统激活/退出
2. 功能激活/降级/退出
3. 驾驶员接管
4. 车道偏离警告
5. 系统故障

**记录内容**:
```json
{
  "timestamp": "2026-03-16T17:49:00Z",
  "event_type": "LANE_DEPARTURE_WARNING",
  "vehicle_speed": 85.5,
  "lateral_offset": 0.35,
  "heading_angle": 3.2,
  "lane_confidence": 0.85,
  "driver_torque": 0.3,
  "system_state": "ACTIVE",
  "warning_level": 1

```

#### 2.6.2 数据存储

- **存储周期**: 30 天循环覆盖
- **存储容量**: ≥ 1000 个事件
- **导出方式**: OBD 接口 / OTA 上传

## 3. 性能要求

| 指标 | 要求 |
|------|------|

## 4. 测试验证

### 4.1 仿真测试

**场景覆盖**:
- 直道保持
- 弯道保持（R=250m, 500m, 1000m）
- 车道切入/切出
- 车道线遮挡/丢失
- 不同天气条件（晴天、雨天、逆光）

**通过标准**: 所有场景通过率 ≥ 98%

### 4.2 实车测试

**测试项目**:
1. **功能验证**: 激活/退出/警告/降级
2. **性能验证**: 横向偏差/响应时间/舒适性
3. **故障验证**: 传感器故障/通信故障
4. **耐久性**: 10,000 km 路试

**测试道路**:
- 高速公路（G2/G3/G15 等）
- 测试里程: ≥ 5,000 km

## 5. 法规要求

### 5.1 C-NCAP 要求

- **测试速度**: 72-80 km/h
- **测试场景**: 车道偏离 + 系统回正
- **通过标准**: 车辆不压线或回到车道内

### 5.2 UN-R79 要求

- **最大力矩**: ≤ 5 Nm（我们设计为 3 Nm）
- **驾驶员优先**: 驾驶员可随时接管
- **故障通知**: 故障时必须有明确通知

### 5.3 功能安全

- **ASIL 等级**: ASIL-B
- **安全目标**: 防止 unintended steering
- **安全机制**: 双路感知、独立监控、故障降级

| 信号 | 来源 | 频率 | 精度 |

| 信号 | 目标 | 频率 | 范围 |

| 状态信息 | 仪表盘 | 10Hz | - |
| 警告信息 | HMI | 事件触发 | - |
| DTC 记录 | 诊断 | 事件触发 | - |
